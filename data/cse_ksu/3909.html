授業概要／Course outline  
インターネットの発展に伴い，クラウドコンピューティングというキーワードが象徴するように，複雑な処理を必
要とする高度なアプリケーションの多くがネットワークを経由して利用されるようになっている．複数の計算機を
利用した大規模計算やネットワークを考慮したセキュリティ対応など，アプリケーションは日々高度化している．
そのため，新たなアプリケーションの開発には，既存のソフトウェアの組み合わせやフレームワークの利用による
迅速な開発技術，そしてブラックボックス化するネットワーク，仮想化インフラ，ミドルウェアなどの特性を分析
し，安定運用する技術が必要となる．本特別研究では，普及しつつあるWebアプリケーションフレームワーク，並
列分散計算フレームワーク等による開発技術の基礎を習得し，いつでもどこでも利用可能な高性能で利便性の高い
Webアプリケーションの開発，大規模計算を必要とするアプリケーションの開発，それを支えるインフラの運用管
理システムの構築などを目指す．
授業内容・授業計画／Course description・plan  
特別研究Iで習得した技術を用いて各自の研究テーマに取り組む．

- クラウドおよびエッジコンピューティングを柔軟に活用可能なData Flow Platformの構築と評価
- エッジコンピューティング環境のためのコンポーネント管理基盤の研究開発
- 観光地における観光流動分析とその応用に関する研究開発
- バスや定点観測センサによるセンシング基盤を活用したスマートシティアプリケーションの実装と評価
- 音によるイベント検知に関する研究開発
- 交通事故を想定した救急活動支援システムの研究開発
- IT/OTシステムの大規模な監視データの蓄積・分析方式の実装と評価
- 構造化オーバレイネットワークと Software Defined Network (SDN) の連携機構の実装と評価
- Software Defined Network (SDN) におけるトラフィック最適化機構の実装と評価
- デジタルサイネージを用いた効果的なセンシング情報提示アプリケーションの実装と評価
- インターネットミドルウェア（OS，ネットワークサーバ，大規模分散システム）におけるパフォーマンスチュー
ニング
- Webアプリケーションフレームワークの調査および応用

授業の到達目標／Expected outcome  
目標とするソフトウェア機能の実現に必要となるオープンソースソフトウェアを調査して，システムを構築するた
めに必要な技術を身につける．また，他の人が同様な問題に取り組む場合に負荷を軽減できるように，調査した内
容をドキュメント化して参照しやすいようにまとめる技術を身につける．
身に付く力／Special abilities to be attained  
論理的思考力（課題発見力、計画力、論理的分析力、総合的判断力）、実践力（働きかけ力、実行力、主体性）、
コミュニケーションスキル（傾聴力）
履修上の注意／Special notes, cautions  
分散システムの構築にはネットワークの知識が必要となるため，「コンピュータネットワークI・II」等でTCP/IP
に関する基礎的な知識を事前に習得しているのが望ましい．また，履修にはプログラミングのスキルが重要となる
ので，プログラミング演習等の内容は各自復習しておく必要がある．
